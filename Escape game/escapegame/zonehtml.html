HTML Enigme

Pour définir le titre principal de votre page, utilisez la balise <h1> :

<h1>Mon Titre Principal</h1>

2. Paragraphes de texte

Pour ajouter des paragraphes de texte, utilisez la balise <p> :

<p>Ceci est mon premier paragraphe.</p>
<p>Et voici mon deuxième paragraphe.</p>

3. Image

Pour insérer une image, utilisez la balise <img> :

<img src="image.png">

Assurez-vous de remplacer "chemin/vers/mon/image.png" par l'URL ou le chemin d'accès de votre image, et "Mon Image" par une description appropriée.

N'oubliez pas de fermer chaque balise avec </> à la fin de chaque élément.

Vous pouvez personnaliser le contenu en remplaçant les exemples par vos propres textes.

Bonne chance avec la création de votre page HTML !

Enigme : Mettez un titre avec une balise <h1>, ajoutez un paragraphe avec le nombre de mots que vous souhaitez et insérez une image nommée "image.png".
